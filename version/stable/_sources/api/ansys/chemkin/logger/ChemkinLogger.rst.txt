.. py:method:: set_level(level: int) -> None

   Set the logger output level.

   Parameters
   ----------
   level : int, {0, 10, 20, 30, 40, 50}
       Output Level of the logger.
       0 = NOTSET
       10 = DEBUG
       20 = INFO
       30 = WARNING
       40 = ERROR
       50 = CRITICAL



.. py:method:: enable_output(stream=None)

       Stream to output the log output to.
